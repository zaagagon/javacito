# TecnoIAProductos - Java JDBC

Proyecto base para la evidencia GA7-220501096-AA2-EV01.

## Requisitos
- Java JDK 17 o superior
- MySQL Server activo
- Visual Studio Code
- Extensión: Extension Pack for Java
- MySQL Connector/J agregado al proyecto

## Base de datos
Ejecute el archivo:

sql/tecnoia.sql

## Configuración de conexión
Revise el archivo:

src/com/tecnoia/conexion/ConexionBD.java

Cambie usuario y clave si es necesario.

## Ejecutar
Abra el archivo:

src/com/tecnoia/app/Principal.java

Luego presione Run Java en Visual Studio Code.

## Operaciones incluidas
- Insertar producto
- Consultar productos
- Actualizar producto
- Eliminar producto
